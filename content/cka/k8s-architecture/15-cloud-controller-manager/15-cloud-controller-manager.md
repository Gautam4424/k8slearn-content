# 1.5 cloud-controller-manager
> Integrates Kubernetes with **cloud provider APIs** (AWS, GCP, Azure, OpenStack, etc.).

**Manages:**

- Node lifecycle (cloud instance metadata, deletion on termination)
- LoadBalancer Service provisioning (creates cloud LBs)
- Route configuration in cloud networking
- Volume provisioning delegation
