# Installation
```bash
# Download
curl -L https://github.com/etcd-io/etcd/releases/download/v3.5.0/etcd-v3.5.0-linux-amd64.tar.gz -o etcd.tar.gz
tar xzf etcd.tar.gz
./etcd   # starts etcd on port 2379 by default
```
